# COMPONENT_SCADA_Pi4
 - utilizing opendnp3 in C++ on a Raspberry Pi 4 to create a outstation and master for TCP polling of various components such as a Keypad, DHT11, HCSR501 Sensor, and Status of Gateway(outstation).
 - Use of ESP32's through ESP-IDF in Vscode used to send data through TCP over 2.4Ghz connection. Utilizng The wifi capabilities and multiple different ip and socket libraries.
